Authentication

All API requests require an Authorization header using the Bearer token scheme. Two key types are accepted:

  • Per-user API key — issued by an FSI administrator per account. Requests are rate-limited and tracked per key.
  • Global service token — for server-to-server integrations. Contact FSI operations to request one.
Authorization: Bearer <your_api_key>

Keys must be kept confidential. If you suspect your key has been compromised, contact your FSI administrator immediately to have it regenerated.

Base URL

https://<your-instance-host>/api

All endpoints are relative to this base. The host matches the domain you use to access the web quote tool.

Rate Limits

API requests are limited to 30 per minute per key by default. Per-user keys are bucketed by account; the global service token is bucketed by IP address. Rate limit headers are included on every response: 

X-RateLimit-Limit: 30
X-RateLimit-Remaining: 27
X-RateLimit-Reset: 1746564720

Exceeding the limit returns HTTP 429 Too Many Requests. Back off and retry after the reset time indicated in the headers.

Endpoints

POST /api/quote

Generate a new freight quote. Returns a persisted quote record with a unique Quote ID.

Request Body (JSON)

Field Type Required Description
quote_type string Yes "Hotshot" or "Air"
origin string Yes Origin 5-digit US ZIP code (e.g. "98001")
destination string Yes Destination 5-digit US ZIP code
weight number Yes Actual shipment weight in pounds
pieces integer No Number of pieces/units. Default: 1
length number No Package length in inches. Used to compute dimensional weight.
width number No Package width in inches.
height number No Package height in inches.
dim_weight number No Pre-calculated dimensional weight in pounds. Provide this instead of length/width/height if you calculate it on your side. Billable weight is max(weight, dim_weight).
accessorials array of strings No List of accessorial service names to add to the quote. Names are case-insensitive. See the Accessorials section below. Example: ["Liftgate", "Residential Delivery"]
user_email string No Email address to associate with the quote record. If omitted, the quote is created anonymously.

Billable Weight Logic

The system automatically selects the higher of actual weight and dimensional weight as the billable weight. Dimensional weight is calculated as (L × W × H) / 139 when dimensions are provided. The response indicates which method was used via the weight_method field.

GET /api/quote/{quote_id}

Retrieve a previously generated quote by its Quote ID (e.g. Q-BCDFGHJ2). Returns the same payload structure as the POST endpoint.

Path Parameter

ParameterDescription
quote_id The Quote ID returned when the quote was created (e.g. Q-BCDFGHJ2)

Response — 200 OK

Same JSON structure as the POST response above.

Response — 404 Not Found

{
  "error":       "Quote not found",
  "remediation": "Confirm the quote_id exists and belongs to this environment, then retry the lookup."
}

Error Responses

All error responses follow the same JSON structure with an error summary and a remediation field containing the recommended fix. 

{
  "error":       "Missing Authorization header.",
  "remediation": "Provide an Authorization header using 'Bearer <your_api_key>' and retry the request."
}
HTTP StatusMeaning
400 Bad RequestInvalid field value (e.g. unknown quote_type)
401 UnauthorizedMissing or malformed Authorization header
403 ForbiddenAPI key is invalid, disabled, or not approved
404 Not FoundQuote ID does not exist
429 Too Many RequestsRate limit exceeded — back off and retry
500 Internal Server ErrorServer-side error — contact FSI operations

Examples

Basic Hotshot Quote

curl -X POST https://<host>/api/quote \
  -H "Authorization: Bearer <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "quote_type":  "Hotshot",
    "origin":      "98001",
    "destination": "90210",
    "weight":      520,
    "pieces":      3
  }'

Air Quote with Dimensions and Accessorials

curl -X POST https://<host>/api/quote \
  -H "Authorization: Bearer <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "quote_type":    "Air",
    "origin":        "10001",
    "destination":   "90210",
    "weight":        150,
    "pieces":        2,
    "length":        48,
    "width":         40,
    "height":        36,
    "accessorials":  ["Liftgate", "Residential Delivery"],
    "user_email":    "dispatcher@example.com"
  }'

Retrieve a Saved Quote

curl https://<host>/api/quote/Q-BCDFGHJ2 \
  -H "Authorization: Bearer <your_api_key>"

Python Example

import requests

API_KEY = "<your_api_key>"
BASE_URL = "https://<host>/api"

response = requests.post(
    f"{BASE_URL}/quote",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={
        "quote_type":   "Hotshot",
        "origin":       "98001",
        "destination":  "90210",
        "weight":       750,
        "pieces":       1,
        "accessorials": ["Liftgate"],
    },
)
response.raise_for_status()
quote = response.json()
print(f"Quote {quote['quote_id']}: ${quote['total']:.2f}")
